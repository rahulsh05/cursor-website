# Cursor Website Clone

This project is a front-page recreation of the Cursor website using HTML and CSS.

## Sections Recreated
- Navigation bar
- Hero section
- Trusted companies section
- Feature sections (Agents, Autocomplete, Integrations)
- Testimonials
- Changelog section
- Footer


## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
